# AfriLION Language Creator: Submission Guide

This guide provides instructions for Language Creators to submit original content for the AfriLION project. Your contributions form the \"Seed Corpus\" for low-resource African languages.

## 📝 Content Guidelines
We require **original, human-written sentences**. Do not use machine translation (Google Translate, etc.).

### Categories
1. **Technical**: News, health, agriculture, or technology topics.
2. **Conversation**: Natural dialogue, greetings, and daily interactions.
3. **Narrative**: Short stories, cultural history, or descriptive passages.

## 🛠 Submission Methods

### Option 1: GitHub (Preferred for Technical Users)
1. Fork the `afrilion` repository.
2. Add your data to `data/creators/[your_name]/[language]_[date].txt`.
3. Submit a Pull Request (PR) with the tag `data: add creator submission`.

### Option 2: Label Studio
1. Log in to the LocaleNLP Label Studio instance (credentials provided by your Regional Coordinator).
2. Select your assigned language project.
3. Type sentences directly into the interface.

## ✅ Quality Checklist
- [ ] **Originality**: Sentences are written by you, not copied from the web.
- [ ] **Script**: Use the correct script for your language (e.g., Ge'ez for Amharic/Tigrinya).
- [ ] **Encoding**: Save files in UTF-8 format.
- [ ] **Diversity**: A mix of technical and conversational content.

## 💰 Compensation
Submissions are audited by Regional Coordinators. Once verified, stipends are processed monthly via the approved payment channel for your region.
